# Car-Free Map — Architecture & Design Decisions

## Concept
A pedestrian-first map that inverts the car-centric default of most maps.

## Color Philosophy
- Green = nature (background, default)
- Warm yellow/brown = urban, human spaces
- Gray = dead/uninteresting (industry, motorways)

## Road Hierarchy
1. Motorway/trunk: "dead space" — gray, dashed when zoomed out, real-width fill when close
2. Primary/secondary: gray solid lines, not inviting
3. Walkable streets (tertiary, residential, etc.): walkability color gradient (gray→yellow→orange)
4. Paths/cycleways: separate thin brown-orange lines, only from z14

## Tech Stack
- MapLibre GL JS in a SvelteKit app
- Vector tiles from OpenFreeMap (OpenMapTiles schema)
- Style JSON generated by `generate_style.py` from `config.yaml`
- Workflow: edit config.yaml → run generator → refresh browser

## Key Constraints
- sprite: "" (empty, no sprite source — don't use icon-image)
- Fonts: Noto Sans Regular/Bold/Italic from OpenFreeMap glyph server
- Real-width streets from z15+ using meter-to-pixel conversion
